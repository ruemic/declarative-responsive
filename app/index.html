Declarative Responsive Design

Build an efficient, robust & highly maintainable presentational layer with a declarative approach to responsive design. Here's how:

  • Leverage the Community - Use an open source responsive framework with a large community behind it. The latest version of Bootstrap or Foundation have solved 1,000's of messy cross-browser edge cases for you.

  • Study Your Framework - Use them in your HTML to get as close as possible to your target look & feel before writing any CSS. To see what I'm talking about: View the source

  • Customize with Variables - Adjust the Sass/Less variables of your framework to suit your needs. For this example, I've bumped up the Bootstrap grid to 36 columns for greater control over the layout:

    $grid-columns: 36;

  • Roll Your Own - When you've reached the limits of what you can achieve with the core framework, consider making your own utility classes to address common style patterns. For example, Bootstrap 3 doesn't offer any vertical spacing utilities, so I've rolled my own:

    /* Vertical Spacing */
.b10 { margin-bottom: 10px; }
.b30 { margin-bottom: 30px; }
.b50 { margin-bottom: 30px; }
